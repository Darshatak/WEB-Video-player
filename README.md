# WEB-Video-player
It has features like Play, Pause, Forward, Backwads, Change speed of video playback, fullscreen mode, change volume.


Keyboard Shortcuts:-

Play/Pause = k
Forward = l / rightarrow
backward = j / leftArrow
FUllScreen = F
Volume Change = UpArrow or DownArrow
