# WEB-Video-player
It has features like Play, Pause, Forward, Backwads, Change speed of video playback, fullscreen mode, change volume.


Keyboard Shortcuts:-

Play/Pause = k <br>
Forward = l / rightarrow <br>
backward = j / leftArrow <br>
FUllScreen = F <br>
Volume Change = UpArrow or DownArrow <br>
